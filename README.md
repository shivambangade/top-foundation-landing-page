# top-foundation-landing-page

This repository contains my solution to **The Odin Project - Foundations Landing Page**.

The main goal of this project was to practice and understand:

- HTML structure and semantic elements
- CSS fundamentals
- Flexbox layouts
- Spacing, alignment, and positioning
- Building a complete landing page from a design reference

I built the project myself while learning the concepts. It isn't a pixel-perfect recreation of the reference design, but it closely follows the layout and helped me gain a solid understanding of Flexbox and page structure.

## Technologies Used

- HTML5
- CSS3
- Flexbox

## What I Learned

- Creating semantic HTML structures
- Using Flexbox to build responsive layouts
- Working with margins, padding, and spacing
- Understanding browser default styles
- Structuring CSS for readability and maintainability

This project is part of **The Odin Project Foundations** curriculum.
